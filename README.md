# Student Task Manager (Node.js)

A simple RESTful Task Management API built using Node.js and Express.

## Features
- Create, read, update, and delete tasks
- JSON-based storage
- REST API structure

## Tech Stack
- JavaScript
- Node.js
- Express.js

## How to Run
1. Clone the repository
2. Run `npm install`
3. Start server using `node app.js`

## Future Improvements
- MongoDB integration
